# Prompts

AI Interaction Prompts Used During Development

This document lists representative prompts used during the development of the CoverLetter AI project.
AI was utilized as a technical assistant to accelerate development, refine architecture, and improve user experience.

- "How to  Add Storybook stories and basic tests."

- "Add a new Storybook story that demonstrates all `Button` variants with hover and disabled states."
- "Update Storybook docs to include usage examples for `Input` and `Card`."
- "Remove unused styles in `app/page.tsx` and consolidate repeated values into constants."
- "Audit `src/globals.css` for unused rules and keep only what is needed for the app."
